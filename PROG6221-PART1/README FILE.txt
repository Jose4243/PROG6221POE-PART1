# Cybersecurity Awareness Chatbot

## Overview
This chatbot educates users about cybersecurity best practices while providing a friendly and interactive experience.

## Features
- Voice greeting upon launch
- ASCII art logo display
- Personalized user interaction
- Basic responses to common cybersecurity questions
- Input validation for better user experience
- Enhanced console UI with visual elements

## Getting Started
1. Clone the repository.
2. Ensure you have the `"C:\Users\contr\source\repos\PROG6221-PART1\PROG6221-PART1\ElevenLabs_2025-03-26T15_56_04_George_pre_s50_sb75_se57_b_m2.wav"` file in the project directory.
3. Compile and run the application.

## Usage
- Enter your name when prompted.
- Ask questions related to cybersecurity, such as "How are you?", "What can you tell me about passwords?", etc.
